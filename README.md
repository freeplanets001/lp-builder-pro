# LP Builder Pro 🚀

ノーコードでプロフェッショナルなランディングページを作成できるビルダーです。

![LP Builder Pro](https://img.shields.io/badge/React-18-blue) ![TypeScript](https://img.shields.io/badge/TypeScript-5-blue) ![Tailwind](https://img.shields.io/badge/Tailwind-3-blue)

## ✨ 機能

- 📱 **14種類のセクション** - ヒーロー、特徴、料金、FAQ、お客様の声など
- 🎨 **スタイルエディター** - 背景色/グラデーション/画像、テキスト色、余白
- 📸 **画像アップロード** - ドラッグ&ドロップ対応
- 📋 **テンプレート** - SaaS、コーチング、EC、ポートフォリオ
- 📤 **HTMLエクスポート** - スタンドアロンHTMLをダウンロード
- 📱 **レスポンシブプレビュー** - PC/タブレット/モバイル表示切替
- ✏️ **インライン編集** - クリックしてテキストを直接編集

## 🛠️ 技術スタック

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## 🚀 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
