# LP Builder Pro 🚀

世界一のノーコードLPビルダー。プロフェッショナルなランディングページを数分で作成。

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react) ![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript) ![Tailwind](https://img.shields.io/badge/Tailwind-3-06B6D4?logo=tailwindcss) ![Vite](https://img.shields.io/badge/Vite-4-646CFF?logo=vite)

## ✨ 機能一覧

### 📐 22種類のセクション
- **メイン**: ヒーロー
- **コンテンツ**: 特徴、料金、FAQ、お客様の声、実績、ロゴ、比較表、タイムライン、チーム紹介、テキストブロック、画像+テキスト
- **メディア**: 動画、ギャラリー、マップ
- **アクション**: CTA、お問い合わせ、カウントダウン
- **レイアウト**: フッター、区切り線、スペース
- **カスタム**: HTML埋め込み

### 🎨 高度なスタイル設定
- **背景**: 単色 / グラデーション / 画像（オーバーレイ対応）
- **タイポグラフィ**: 33種類のGoogle Fonts、サイズ/太さ/行間/字間
- **余白**: パディング・マージン（上下左右個別設定）
- **効果**: ボーダー、角丸、シャドウ
- **アニメーション**: 15種類（フェード、ズーム、バウンス、スライド等）
- **レイアウト**: 幅（全幅/標準/狭い）、カラム数、間隔、縦位置

### 🌈 グローバルスタイル
- カラーパレット（8種類のプリセット）
- メイン/サブ/アクセントカラー
- 見出し/本文フォント
- 角丸・間隔の一括設定

### 📱 レスポンシブ対応
- デスクトップ/タブレット/モバイルプレビュー
- デバイス別の表示/非表示設定

### 🔄 便利機能
- Undo/Redo（50回まで）
- JSON保存/読み込み
- HTMLエクスポート（スタンドアロン）
- 6種類のテンプレート
- インライン編集
