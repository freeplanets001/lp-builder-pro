# LP Builder Pro 🚀

**世界一のランディングページビルダー** - ノーコードでプロフェッショナルなLPを作成

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react) ![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript) ![Tailwind](https://img.shields.io/badge/Tailwind-3-06B6D4?logo=tailwindcss) ![Vite](https://img.shields.io/badge/Vite-4-646CFF?logo=vite)

## ✨ 機能一覧

### 🎨 デザイン
- **15種類のセクション** - ヒーロー、特徴、料金、FAQ、お客様の声、ギャラリーなど
- **高度なスタイリング** - 背景（単色/グラデーション/画像）、ボーダー、シャドウ、角丸
- **タイポグラフィ** - Google Fonts対応、サイズ、行間、文字間隔
- **カラーシステム** - テーマカラー、カスタムカラー

### 📐 レイアウト
- **グリッドシステム** - 1〜6列のカラム設定
- **余白調整** - 上下左右を個別に設定
- **配置** - 左/中央/右揃え
- **最大幅** - 狭い〜全幅まで選択可能

### ⚡ アニメーション
- **12種類のアニメーション** - フェードイン、スライド、ズーム、バウンスなど
- **タイミング制御** - 持続時間、遅延を調整

### 🔘 ボタンカスタマイズ
- **形状** - 角丸、ピル型、角形
- **ホバーエフェクト** - 浮き上がり、拡大、光る、色反転
- **完全カスタマイズ** - 色、サイズ、パディング

### 🛠️ 高度な機能
- **Undo/Redo** - 操作を元に戻す/やり直す
- **カスタムCSS** - セクションごと/グローバルでCSS追加
- **SEO設定** - タイトル、説明、OGP画像
- **HTMLエクスポート** - スタンドアロンHTMLをダウンロード
- **レスポンシブプレビュー** - PC/タブレット/モバイル表示確認

### 📋 テンプレート
- **SaaS** - ソフトウェアサービス向け
- **コーチング** - 個人サービス向け
- 今後追加予定...

## 🚀 使い方

```bash
# インストール
npm install

# 開発サーバー
npm run dev

# ビルド
npm run build
```
